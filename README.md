## 📖 Project Overview

Data ingestion is one of the most critical stages of any CRM workflow. **GrowEasy AI CSV Importer** eliminates the complexity of processing unstructured CSV files by leveraging **Google Gemini AI** to intelligently identify, understand, and map arbitrary column headers into standardized CRM fields.

The application combines AI-powered data extraction, strict validation rules, enterprise-grade batch processing, and a resilient frontend architecture to deliver reliable, accurate, and scalable CRM data ingestion even during API failures or network interruptions.

## ✨ Key Features & Architecture

### 🧠 Intelligent AI Column Mapping

Google Gemini AI automatically understands arbitrary CSV structures and accurately maps unknown column names into predefined CRM fields.

Example:

```
Contact No.
        ↓
mobile_without_country_code
```

---

### ⚙️ Enterprise Batch Processing

Large CSV datasets are intelligently divided into optimized batches, preventing AI hallucinations, syntax inconsistencies, timeout issues, and API rate-limit failures while ensuring scalable processing.

---

### 🛡️ Strict CRM Validation

Every extracted record undergoes backend validation before being accepted.

The validation engine automatically removes records missing essential CRM information such as:

- Name
- Email
- Phone Number

ensuring only high-quality data is imported.

---

### ⚡ Resilient Frontend Architecture

A built-in fallback mechanism guarantees uninterrupted operation.

Whenever the AI service or backend becomes temporarily unavailable, the application gracefully switches to local processing, ensuring a seamless user experience without data loss.

---

### 🎨 Modern User Experience

Built using **Next.js**, **TypeScript**, and **Tailwind CSS**, the application provides:

- Drag-and-drop CSV uploads
- Intelligent data preview
- AI-powered CRM field mapping
- Live processing indicators
- Responsive data tables
- Dark mode support
- Clean and modern interface

# 🚀 Quick Start

## Prerequisites

- Node.js **v18** or later
- Google Gemini API Key from Google AI Studio

---

### 1️⃣ Clone Repository

```bash
git clone https://github.com/anushkagupta47/GrowEasy-AI-CSV-Importer.git

cd GrowEasy-AI-CSV-Importer
```

---

### 2️⃣ Backend Setup

```bash
cd Backend

npm install
```

Create a `.env` file.

```env
GEMINI_API_KEY=your_actual_api_key_here

PORT=5000
```

Start the backend.

```bash
node server.js
```

---

### 3️⃣ Frontend Setup

Open another terminal.

```bash
cd frontend

npm install

npm run dev
```

Visit

```
http://localhost:3000
```
